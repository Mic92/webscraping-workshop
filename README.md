# Webscraping-Workshop

Folien und Code zum Webscraping workshop auf den [Datenspuren 2015](https://www.datenspuren.de/2015/index.html)

## Vorbereitung

### Ubuntu/Debian

```
$ sudo apt-get install python3-bs4 python3-requests
```
